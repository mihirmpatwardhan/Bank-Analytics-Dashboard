# 🏦 Bank Analytics Dashboard

## 🚀 Project Overview

The **Bank Analytics Dashboard** is a comprehensive and interactive Power BI solution designed to transform raw banking data into meaningful business insights. This project focuses on analyzing financial operations, customer behavior, loan distribution, and transaction trends through visually rich and dynamic dashboards.

The dashboard enables stakeholders to monitor key performance indicators (KPIs), identify patterns, and make data-driven decisions efficiently.

---

## 🎯 Objectives

* To analyze banking transactions and financial performance
* To understand customer demographics and behavior
* To evaluate loan distribution based on credit scores
* To forecast future transaction trends
* To provide an interactive decision-support system

---

## 📊 Key Insights Delivered

* 📌 Identification of **high-performing account types** based on transaction volume
* 💰 Analysis of **total loan amount vs credit score categories**
* 📈 Visualization of **transaction trends and future forecasts**
* 👥 Understanding **customer distribution across cities and employment sectors**
* 🧠 Evaluation of **average credit score patterns**
* 🔍 Monitoring of **account opening trends over time**

---

## 🧩 Dashboard Features

### 🔹 KPI Metrics

* Total Transactions
* Total Loan Amount
* Total Account Balance
* Average Credit Score

### 🔹 Interactive Visualizations

* Transaction Count by Account Type
* Loan Analysis (Credit Score vs Loan Amount vs Interest Rate)
* Account Opening Status (Quarter-wise trends)
* Customer Demographics (City-wise & Sector-wise distribution)
* Transaction Forecast (Predictive trend analysis)
* Customer Loyalty Distribution

### 🔹 Filters & Slicers

* Year Selection (2023, 2024, 2025)
* Currency (EUR, GBP, USD, JPY)
* Account Type (Checking, Savings, Credit, Loan)
* Branch Code

## 🛠️ Tools & Technologies Used

* **Power BI** – Data modeling and dashboard creation
* **Microsoft Excel / CSV** – Data preprocessing and storage
* **DAX (Data Analysis Expressions)** – Custom measures and calculations
* **Power Query** – Data transformation and cleaning

## 🧠 Technical Implementation

* Data cleaning and transformation using Power Query
* Creation of calculated columns and measures using DAX
* Implementation of interactive slicers and filters
* Use of advanced visuals for storytelling
* Forecasting techniques applied for trend prediction
* Optimized dashboard layout for better user experience
